# מדריך הסגנון של יעל

> קובץ זה ריק כרגע — ממתין להגדרות סגנון מהמשתמש.

כשהמשתמש יוסיף כאן הנחיות סגנון, יעל תקרא אותן בתחילת כל משימה ותכתוב לפיהן.

---

## מה להגדיר כאן

- **טון הדיבור** — רשמי / שיחתי / מקצועי / חברותי
- **אורך משפטים** — קצרים וקולעים / ארוכים ומורחבים
- **מבנה מאמר** — כותרת, תת-כותרות, פסקאות, סיכום
- **מילים שמשתמשים בהן / נמנעים מהן**
- **שפה** — עברית / אנגלית / דו-לשוני
- **פורמט** — bullet points / פסקאות רציפות / שילוב
